This project aims at the vizualization of the featuring network of artist for different country. Each node represent an artist an a link is an existing feat between two artist. The bigger the node the more connection it has. Colouring are obtained by "modularity" classification. This project was realized mainly in python, the data come from link and visualization realized inside Gephi. The following database are available:

  • Belgium artists
  • French artists
  • All the previous databases unified.
